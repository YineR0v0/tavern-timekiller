
# Tavern Timekiller

一个专为 SillyTavern 设计的悬浮窗小游戏集合扩展。包含像素农场、2048、扫雷等，让你在等待 AI 回复时消磨时间。

## 功能
- **像素农场**：种植、浇水、收割，赚取金币升级。
- **悬浮窗**：可拖动、可最小化，不影响酒馆正常操作。
- **启动球**：右下角常驻 🌱 按钮，防止窗口丢失。
- **自动存档**：游戏进度自动保存在本地。

## 安装方法

1. **自动安装 (推荐)**:
   - 打开 SillyTavern。
   - 进入 **Extensions (扩展)** 菜单。
   - 点击 **Install Extension**。
   - 输入本仓库的 GitHub URL。
   - 点击 **Install**。

2. **手动安装**:
   - 下载本仓库代码。
   - 将文件夹放入 SillyTavern 的 `public/scripts/extensions/` 目录下。
   - 重启 SillyTavern。

## 使用
- 刷新酒馆页面后，屏幕右下角会出现一个 🌱 按钮。
- 点击按钮即可打开/隐藏游戏窗口。
